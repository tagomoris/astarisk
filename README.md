# Astarisk

Astarisk is a tool to visualize `RubyVM::AbstractSyntaxTree` nodes on console.

That is named as "AST a risk".

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'astarisk'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install astarisk
